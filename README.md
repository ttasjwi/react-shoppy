
# React-Shoppy

- 설명 : 드림코딩 엘리님의 React 강의 학습
- 학습 목적 : 프론트 엔드 개발자가 아니지만 기본적인 프론트엔드 전반적인 흐름을 이해하고 싶었음

---

# 로컬에서 프로젝트 실행
```shell
# 가져오기
git clone https://github.com/ttasjwi/react-shoppy

cd react-shoppy

# 루트 경로에서
yarn install
yarn start
```
